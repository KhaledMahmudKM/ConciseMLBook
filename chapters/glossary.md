# Glossary

## A

**Accuracy**  
The proportion of predictions that are correct.

**Activation Function**  
A function that introduces non-linearity into a neural network.

**AdaBoost**  
An ensemble learning algorithm that combines many weak learners into a stronger model.

**Agent**  
In reinforcement learning, the entity that learns by interacting with an environment.

**Algorithm**  
A step-by-step procedure used to solve a computational problem.

**Artificial Intelligence (AI)**  
The field of creating systems capable of performing tasks that typically require human intelligence.

**Attention Mechanism**  
A neural network technique that enables a model to focus on the most relevant parts of the input.

**Autoencoder**  
A neural network trained to compress data into a smaller representation and reconstruct it.

## B

**Backpropagation**  
The algorithm used to compute gradients and update neural network weights.

**Bagging**  
An ensemble method that trains multiple models on different subsets of data and combines their predictions.

**Batch**  
A subset of the training data processed in one iteration.

**Batch Normalization**  
A technique that normalizes activations during training to improve convergence.

**Batch Size**  
The number of training samples processed before updating model parameters.

**Bayesian Learning**  
A learning approach based on Bayes' theorem for updating probabilities as new evidence becomes available.

**Bias**
- A learnable parameter added to the weighted sum of inputs in a neural network.
- The error introduced by simplifying assumptions in a model.
- Systematic unfairness in model predictions.

**Bias-Variance Tradeoff**  
The balance between underfitting and overfitting.

**Binary Classification**  
A classification problem with only two possible classes.

**Boosting**  
An ensemble technique that sequentially trains models to correct previous errors.

## C

**Categorical Data**  
Data represented by categories rather than numerical values.

**Classification**  
Predicting one of several predefined classes.

**Class Imbalance**  
A situation where some classes have far more samples than others.

**Clustering**  
Grouping similar data without predefined labels.

**Confusion Matrix**  
A table summarizing classification performance.

**Convolution**  
A mathematical operation used in convolutional neural networks to extract features.

**Convolutional Neural Network (CNN)**  
A neural network specialized for image and spatial data.

**Cost Function**  
A function measuring prediction error during optimization.

**Cross Validation**  
A technique for evaluating a model by repeatedly splitting data into training and validation sets.

## D

**Data Augmentation**  
Creating additional training examples by modifying existing ones.

**Data Leakage**  
Using information during training that would not be available during prediction.

**Dataset**  
A collection of data samples.

**Decision Boundary**  
The boundary separating different predicted classes.

**Decision Tree**  
A tree-structured model that makes decisions based on feature values.

**Deep Learning**  
Machine learning based on neural networks with many layers.

**Dimensionality**  
The number of features describing each sample.

**Dimensionality Reduction**  
Reducing the number of features while preserving useful information.

**Dropout**  
A regularization technique that randomly disables neurons during training.

## E

**Embedding**  
A dense numerical representation of data such as words or images.

**Ensemble Learning**  
Combining multiple models to improve performance.

**Environment**  
In reinforcement learning, the system with which the agent interacts.

**Epoch**  
One complete pass through the training dataset.

**Error**  
The difference between a prediction and the true value.

**Evaluation**  
Measuring model performance.

**Explainability**  
The ability to understand how a model makes predictions.

## F

**False Negative (FN)**  
A positive sample incorrectly predicted as negative.

**False Positive (FP)**  
A negative sample incorrectly predicted as positive.

**Feature**  
An individual measurable property of the data.

**Feature Engineering**  
Creating useful features from raw data.

**Feature Extraction**  
Automatically deriving informative features from data.

**Feature Scaling**  
Adjusting feature values to a common range.

**Feature Selection**  
Selecting the most useful features for training.

**Fine-Tuning**  
Further training a pre-trained model on a new task.

**F1 Score**  
The harmonic mean of precision and recall.

**Forward Propagation**  
Passing input data through a neural network to produce predictions.

## G

**GAN (Generative Adversarial Network)**  
A model consisting of a generator and discriminator trained together.

**Generalization**  
A model's ability to perform well on unseen data.

**Gradient**  
The slope of the loss function.

**Gradient Descent**  
An optimization algorithm that minimizes the loss function.

**Grid Search**  
A method for systematically testing combinations of hyperparameters.

## H

**Hidden Layer**  
A layer between the input and output layers of a neural network.

**Hyperparameter**  
A configuration value chosen before training.

## I

**Image Classification**  
Assigning labels to images.

**Inference**  
Using a trained model to make predictions.

**Input Layer**  
The first layer of a neural network.

## K

**K-Means**  
A clustering algorithm that partitions data into k groups.

**K-Nearest Neighbors (KNN)**  
An algorithm that predicts based on nearby training examples.

## L

**Label**  
The correct output associated with a training sample.

**Learning Rate**  
The size of each optimization step.

**Linear Regression**  
A regression model that fits a straight line.

**Logistic Regression**  
A classification algorithm that predicts probabilities.

**Long Short-Term Memory (LSTM)**  
A recurrent neural network designed to model long-term dependencies.

**Loss Function**  
A function measuring prediction error.

## M

**Machine Learning (ML)**  
Algorithms that learn patterns from data.

**Mini-Batch**  
A small subset of training data processed in one iteration.

**Model**  
A mathematical representation learned from data.

**Multiclass Classification**  
Classification involving more than two classes.

## N

**Natural Language Processing (NLP)**  
Machine learning techniques for processing human language.

**Neural Network**  
A network of interconnected computational units inspired by the brain.

**Neuron**  
A computational unit in a neural network.

**Normalization**  
Scaling numerical data to a common range.

## O

**One-Hot Encoding**  
Representing categories as binary vectors.

**Optimization**  
Finding parameter values that minimize the loss.

**Optimizer**  
An algorithm used to update model parameters.

**Output Layer**  
The final layer that produces predictions.

**Overfitting**  
Learning the training data too closely, including its noise.

## P

**Parameter**  
A value learned during training.

**PCA (Principal Component Analysis)**  
A technique for reducing dimensionality while preserving variance.

**Policy**  
In reinforcement learning, the strategy an agent follows.

**Pooling**  
Reducing the size of feature maps in a CNN.

**Precision**  
The proportion of predicted positives that are correct.

**Prediction**  
The output produced by a trained model.

**Preprocessing**  
Preparing raw data before training.

## Q

**Q-Learning**  
A reinforcement learning algorithm that learns the value of actions.

## R

**Random Forest**  
An ensemble of decision trees.

**Recall**  
The proportion of actual positives correctly identified.

**Regression**  
Predicting continuous numerical values.

**Regularization**  
Techniques for reducing overfitting.

**Reinforcement Learning**  
Learning through interaction with an environment using rewards.

**Recurrent Neural Network (RNN)**  
A neural network designed for sequential data.

**Reward**  
A numerical signal indicating the success of an action.

## S

**Sigmoid Function**  
An activation function producing values between 0 and 1.

**Softmax**  
An activation function that converts outputs into probabilities.

**Support Vector Machine (SVM)**  
A classification algorithm that finds the optimal separating boundary.

**Supervised Learning**  
Learning from labeled examples.

## T

**Test Set**  
Data used to evaluate the final model.

**Training**  
The process of learning from data.

**Training Set**  
The portion of data used to train a model.

**Transformer**  
A neural network architecture based on attention mechanisms.

**Transfer Learning**  
Adapting a pre-trained model to a new task.

**True Negative (TN)**  
A negative sample correctly classified as negative.

**True Positive (TP)**  
A positive sample correctly classified as positive.

## U

**Underfitting**  
A model that is too simple to capture patterns in the data.

**Unsupervised Learning**  
Learning from unlabeled data.

## V

**Validation Set**  
Data used during training to tune hyperparameters.

**Variance**  
The sensitivity of a model to changes in the training data.

## W

**Weight**  
A learnable value that determines the influence of an input.

**Weak Learner**  
A model that performs only slightly better than random guessing.

## X

**XGBoost**  
An efficient and widely used gradient boosting algorithm.

## Z

**Zero-Shot Learning**  
A technique where a model performs tasks without task-specific training examples.
